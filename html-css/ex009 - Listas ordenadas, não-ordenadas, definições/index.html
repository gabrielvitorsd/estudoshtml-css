1. Trabalhando com listas

1.1 Listas ordenadas

Normalmente ocorre uma lista ter que mudar a numeração. Igual neste caso a seguir:
1 - Acordar
2 - Escovar os dentes
2 - Ligar para a mãe
3 - Tomar café
4 - Lavar roupa

Para corrigir esse problema. Utilizaremos a TAG - OL - ORDERLIST.

  2. Escovar os dentes
  3. Ligar para a mãe
  4. Tomar café
